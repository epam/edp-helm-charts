## Add The Repository

```bash
helm repo add epamedp https://epam.github.io/edp-helm-charts/stable
helm repo update
```

or

```bash
helm repo add epamedp-snapshot https://epam.github.io/edp-helm-charts/snapshot
helm repo update
```
